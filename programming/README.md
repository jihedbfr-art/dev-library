## How to learn a new language fast

1. Data types, variables, control flow, functions — 1 day.
2. How it does collections, errors, and modules — the daily 80%.
3. Its **idioms** — every language has a "right way"; write it like a native, not like your last language.
4. Its tooling: package manager, formatter, test runner, linter.
5. Build something real. Reading tutorials forever teaches nothing.
