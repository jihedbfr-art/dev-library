# About this repo

I started this because I was tired of having my notes scattered across five different tools — a Notion page here, a bookmark folder there, half-finished gists nobody could find again. Every time I hit the same Kubernetes probe bug or forgot the JWT claim I needed to check, I'd re-google it from scratch.

So: one repo, organized properly, public so it's actually useful to more than just me.

I build stuff mostly in the Spring Boot / Angular world (see [notes-app-microservices](https://github.com/jihedbfr-art) if you want to see the code behind some of the backend/microservices pages here — that project is where half of the `backend/microservices` content actually got tested, not just read about). The DevSecOps and cybersecurity sections come from wanting to stop being the guy who ships first and secures later. The telecom module exists because I think it's a criminally under-documented field for developers — everyone explains 5G in marketing terms, nobody explains what a UPF actually does.

## What this is not

- Not a course. No guarantee of a clean learning curve — some pages are deep, some are stubs, some I'll rewrite when I understand the topic better myself.
- Not vendor-neutral in a fake way. When I think a tool is better, I say so.
- Not finished. It never will be — that's the point of a living reference instead of a book.

## What I actually use this for

Honestly? I search it before I search Google now for anything I've already written down. If a page here saves you the same 20 minutes it saved me, it did its job.

— Jihed
